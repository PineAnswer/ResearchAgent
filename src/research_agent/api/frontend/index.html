        Quick start

        欢迎使用论文研读工作台

        从研究问题到证据综述，只需按下面的流程操作。

        1. 新建研究填写主题、研究问题和检索范围，启动文献检索。
        2. 审核候选勾选相关论文；结果不足时可补充关键词或 DOI。
        3. 继续研究确认候选后，让智能体完成论文精读、结果整理和综述写作。
        4. 查看成果在项目页面阅读综述，并通过“研究记录”查看过程与产物。
        5. 管理文献在文献库中导入、整理、比较论文，并围绕文献提问。

        稍后可从右上角“更多工具 → 使用说明”再次打开。
